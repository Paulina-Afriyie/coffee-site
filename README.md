# ☕ Coffee Shop Landing Page

## 📌 Project Overview

This project is a modern **Coffee Shop Landing Page** created using **HTML and CSS**.
The website promotes a coffee brand and provides information about the coffee shop, its features, and a contact form for customers.

The goal of the project is to practice **web design and development concepts**, including semantic HTML structure, responsive layout, and modern CSS styling.

---

## 🚀 Features

* Responsive **Navigation Bar**
* Attractive **Hero Section**
* **Feature Cards** displaying coffee shop benefits
* Professional **Contact Form**
* Clean **Footer Section**
* Modern typography using **Google Fonts**
* Responsive layout suitable for mobile and desktop screens

---

## 🛠 Technologies Used

* HTML5
* CSS3
* Google Fonts
